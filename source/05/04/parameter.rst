Param
=====

計算条件 (もしくは格子生成条件) の入力ダイアログに表示するラベルの定義情報を保持します。

例
----

.. code-block:: xml
   :caption: Parameter の定義例
   :name: ref_Parameter_example
   :linenos:

   <Parameter caption="Time" valueType="real" />

定義例は :ref:`cc_widget_example_func` も参照して下さい。
