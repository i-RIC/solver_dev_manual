.. _iriclib_open_cgns:

Opening a CGNS file
======================

Open a CGNS file.

When opening a CGMS file for modifying, specify IRIC_MODE_MODIFY as mode.

When opening a CGMS file for reading only, specify IRIC_MODE_READ as mode.
