iric_check_cancel
===================

ユーザがソルバーをキャンセルしたか確認する。

形式 (FORTRAN)
---------------
.. code-block:: fortran

   call iric_check_cancel(canceled)

形式 (C/C++)
---------------
.. code-block:: c

   canceled = iRIC_Check_Cancel();

形式 (Python)
---------------
.. code-block:: python

   canceled = iRIC_Check_Cancel()
