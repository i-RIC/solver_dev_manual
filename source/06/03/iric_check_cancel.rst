iric_check_cancel
===================

Checks whether user canceled solver execution.

Format (FORTRAN)
------------------
.. code-block:: fortran

   call iric_check_cancel(canceled)

Format (C/C++)
----------------
.. code-block:: c

   iRIC_Check_Cancel(&canceled);

Format (Python)
----------------
.. code-block:: python

   canceled = iRIC_Check_Cancel()
