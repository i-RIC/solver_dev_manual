cg_iric_flush
===============

計算結果の出力をファイルに書き込む。

形式 (FORTRAN)
---------------
.. code-block:: fortran

   call cg_iric_flush(filename, fin, ier)

形式 (C/C++)
---------------
.. code-block:: c

   ier = cg_iRIC_Flush(filename, fin);

形式 (Python)
---------------
.. code-block:: python

   fin = cg_iRIC_Flush(filename, fin)
