cg_iric_read_complex_count
============================

---------------
.. code-block:: fortran

   call cg_iric_read_complex_count(type, num, ier)

形式 (C/C++)
---------------
.. code-block:: c

   ier = cg_iRIC_Read_Complex_Count(type, &num);

形式 (Python)
---------------
.. code-block:: python

   num = cg_iRIC_Read_Complex_Count(type)
