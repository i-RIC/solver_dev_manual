cg_iric_read_grid_functionaltimesize

---------------

   call cg_iric_read_grid_functionaltimesize(fid, label, count, ier)

.. code-block:: c

   ier = cg_iRIC_Read_Grid_F_functionalTimeSize(fid, label, &count);
