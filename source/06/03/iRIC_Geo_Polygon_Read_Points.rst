iRIC_Geo_Polygon_Read_Points
============================

ポリゴンの頂点の座標を読み込む。

形式 (FORTRAN)
-----------------

.. code-block:: fortran

   call iRIC_Geo_Polygon_Read_Points(geo_handle, x_arr, y_arr, ier)

形式 (C/C++)
-----------------

.. code-block:: c

   ier = iRIC_Geo_Polygon_Read_Points(geo_handle, x_arr, y_arr)

形式 (Python)
-----------------

.. code-block:: python

   x_arr, y_arr = iRIC_Geo_Polygon_Read_Points(geo_handle)
