cg_iric_read_realsingle_f
=========================

CGNSファイルから単精度の実数型の計算条件・格子生成条件の値を読み込む。

形式 (FORTRAN)
---------------
.. code-block:: fortran

   call cg_iric_read_realsingle_f(label, realvalue, ier)

形式 (C/C++)
---------------
.. code-block:: c

   ier = cg_iRIC_Read_RealSingle(label, &realvalue);

形式 (Python)
---------------

Python にはこの関数は存在しない
