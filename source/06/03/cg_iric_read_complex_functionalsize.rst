cg_iric_read_complex_functionalsize
=====================================

複合型格子属性の、関数型の条件の変数のサイズを読み込む。

形式 (FORTRAN)
---------------
.. code-block:: fortran

   call cg_iric_read_complex_functionalsize(fid, type, num, name, size, ier)

形式 (C/C++)
---------------
.. code-block:: c

   ier = cg_iric_read_complex_functionalsize(fid, type, num, name, &size);

形式 (Python)
---------------

Python にはこの関数は存在しない
